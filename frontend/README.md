# Frontend

This is the Next.js frontend for CirclePact.

## Scripts
- `npm run dev` — start the local development server
- `npm run build` — create a production build
- `npm run test` — run the Vitest suite
- `npm run lint` — run ESLint

## Verification
- Production build: verified successfully with `npm run build`
- Test suite: verified successfully with `npm test`
- Analytics and feedback UI: included in the main app experience
